# Ясно — концепт-редизайн: лендинг + воронка подбора психолога

Прототип для демонстрации трёх growth-гипотез. Не является официальным сайтом
Ясно: все данные — mock, аналитика — заглушка с логом в консоль.

## Запуск

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production-сборка без ошибок TypeScript
```

Стек: Vite · React 18 · TypeScript (strict) · Tailwind CSS v4 · motion · lucide-react.
Без UI-китов — весь визуал кастомный. Шрифты: Piazzolla (display) + Onest (UI), Google Fonts.

## Три гипотезы — как проверить кликами

### 1. Сепарация выбора и оплаты — ценность до регистрации
Квиз → подборка → слот → и только потом регистрация (email + телефон, без пароля).

Путь: «Подобрать психолога» (или чип эмоции в hero) → 7 шагов → анимация
подбора (скоринг реально выполняется, `lib/matching.ts`) → 3 карточки
с бейджем «Совпадение N%» и блоком «Почему именно она/он» → «Выбрать время»
→ слот-пикер → **SignupGate** «Почти готово» → «Запись закреплена».
С карточки психолога «Записаться» ведёт сразу в слот-пикер.

### 2. Тесты как acquisition-воронка
Секция «Мини-тесты» (#tests): «Уровень тревоги» (GAD-2) · «Выгорание» ·
«Подойдёт ли мне терапия?». Результат сразу, без email-гейта, с дисклеймером
«не является диагнозом». CTA результата открывает квиз с уже выбранной темой
(шаг 2 предзаполнен) — как и чипы эмоций в hero.

### 3. Активация второй сессии как LTV-рычаг
Экран подтверждения: «что будет на первой сессии», «Добавить в календарь»
(генерация .ics на лету), и мягкий офер «закрепить ритм» с опорой на «81%
чувствуют результат к 5-й сессии». Принятие — тост «Слот предварительно
удержан на 24 часа»; отказ — «Решу после первой сессии», без давления.

## Аналитика

`lib/analytics.ts` — `track(event, props)` → `console.info('[track]', …)`.
Инструментирована вся воронка: `cta_click`, `chip_click`, `quiz_start/step/back/abandon/complete`,
`match_shown`, `therapist_expand`, `slot_selected`, `signup_shown/submitted`,
`booking_confirmed`, `second_session_offer_shown/accepted`, `test_start/complete`,
`breath_start/complete`, `email_capture`. Откройте консоль и пройдите воронку.

## Качество

- `npm run build` — без ошибок и предупреждений TypeScript (strict).
- Адаптив от 360px: грид психологов 3→2→1, квиз и sticky CTA проверены на 375px.
- Доступность: один h1, семантика, alt у фото, видимый фокус (`--sky`),
  квиз управляется с клавиатуры (стрелки/Enter/Esc), жёлтый только с тёмным текстом.
- `prefers-reduced-motion`: декоративные анимации отключаются, дыхательная
  практика переходит на текстовый таймер.
- Никаких тёмных паттернов: без фейковых таймеров и «осталось 2 места» —
  только честная доступность слотов.
