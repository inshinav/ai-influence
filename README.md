# Я-Ферма — AI creator OS для бережного роста «Ясно»

Интерактивный концепт growth-гипотезы для сервиса онлайн-психотерапии «Ясно»:
собственная сеть AI-инфлюенсеров, которые публикуют бережный вертикальный контент
о тревоге, отношениях, выгорании, самооценке и границах, собирают релевантную
аудиторию в TikTok и мягко знакомят её с «Ясно».

Это продуктовая презентация, а не лендинг: посетитель садится за консоль спокойной
медиа-студии и видит, как система думает, генерирует ролик и сама себя проверяет.

> **Рамка концепта.** Это не официальный сайт «Ясно». AI-авторы — не психологи.
> Никаких медицинских обещаний, диагнозов, манипуляций уязвимостью и тёмных
> паттернов. TikTok подаётся как гипотеза роста для проверки на пилоте. Все
> интерактивные функции — локальная симуляция без внешних API.

## Запуск

```bash
npm install
npm run dev      # http://localhost:5173/ai-influence/
npm run build    # tsc + vite, без ошибок и предупреждений
npm run lint     # ESLint, должен быть чистым
```

`base` в `vite.config.ts` — `/ai-influence/` (сайт живёт в подпапке). Для другого
пути: `VITE_BASE_PATH=/ npm run build`.

## Стек

Vite + React 18 + TypeScript (strict) + Tailwind v4 + motion v12 (`motion/react`)
+ lucide-react. Без UI-китов — весь интерфейс кастомный. Шрифты самохостятся
(fontsource): **Geologica** (display), **Inter** (body), **Caveat** (рукописный
акцент). Дизайн-токены (палитра, шкала радиусов и теней, типографика через
`clamp()`) живут в `:root` в `src/index.css`.

## Как менять контент

Весь контент — в **`src/data/aiFarm.ts`** (единственный источник правды). Меняете
там — интерфейс перестраивается сам: счётчики, графики и бейджи выводятся из данных.

- **Авторы** — массив `personas` (имя, `@handle`, цвет, темы, форматы, обязательный
  AI-дисклеймер `disclosure`).
- **Ролики** — массив `clips` (хук, биты, числовые метрики `retention/saves/click`,
  структурные `safetyChecks`, имя видеофайла).
- **Метрики и счётчики** — `system`, `dashboard`, `pilotTrend`, `economicsCurve`
  выводят счётчики, спарклайны и графики; жёстко зашитых чисел в компонентах нет.
- **Тексты секций** — объект `sections` (эйбрау, заголовок, лид каждой секции).
- **Видео** — положите вертикальные MP4 в `public/examples/` с теми же именами, что
  в `clips[].videoFile` — превью телефонов начнут проигрывать живое видео; иначе
  показывается сгенерированная синтетика.

Компоненты секций — в `src/components/`, общая «витрина телефона» — `Phone.tsx`,
хелперы (генерация черновика, формат чисел, пресеты движения, считалка) — в `src/lib/`.

## Этическая рамка — в данных и типах

Безопасность не декларация, а часть модели:

- `disclosure` у каждого автора **обязателен** и выводится видимым бейджем
  (`AiPill`) на каждом появлении автора.
- У роликов — структурные `safetyChecks: SafetyTag[]`; у тем — список `forbidden`
  формулировок.
- Генератор «Консоли» (`src/lib/console.ts`) прогоняет черновик через санитайзер:
  запрещённые формулировки физически не попадают в выдачу, а safety-гейт честно
  отражает применённые ограничения.
- Никаких таймеров, «осталось N мест», exit-попапов и эксплуатации уязвимости.

## Доступность и движение

Один `h1`, семантика, видимый фокус, навигация с клавиатуры. Декоративные анимации
— только `transform/opacity/filter`; при `prefers-reduced-motion` они отключаются,
а стриминг-генерация и считалки показывают финальное состояние сразу.

## Деплой

Содержимое `dist/` (включая `.htaccess` для SPA-fallback) → подпапка хостинга.
