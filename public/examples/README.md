# Video examples

Drop vertical MP4/WebM files here to replace the simulated phone previews.

The current demo looks for these filenames:

- `mira-morning-noise.mp4`
- `nora-reply-later.mp4`
- `lev-strong-person.mp4`
- `sasha-not-perfect.mp4`
- `toma-quiet-no.mp4`
- `kai-crowded-alone.mp4`

If a file is missing, the site shows a generated local placeholder instead. To change names, edit
`src/data/aiFarm.ts`.
